# InvesteBolsaValores

### Requisitos:

 * Python 3
 * NumPy
 * Pandas
 * Quandl
 * BeautifulSoup 4
 * Plotly
 * Django

### Para rodar o programa:
 1. Navegue para: 
 ```shell
 /InvesteBolsaValores/site_bolsa
 ```
 2. Execute: 
 ```shell
 python manage.py runserver
 ```
 3. No navegador acesse: http://127.0.0.1:8000/site_functions/
